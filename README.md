VietNamese:
SONA là ứng dụng nghe nhạc offline trên Android, cho phép người dùng:
 - Phát nhạc từ bộ nhớ thiết bị
 - Xem lời bài hát (lyrics)
 - Gợi ý nhạc thông minh (For You)
 - Hiển thị xu hướng (Trending theo lượt nghe)
 - Quản lý playlist cá nhân
Ứng dụng sử dụng kiến trúc hiện đại với Media3 + MediaSessionService, đảm bảo trải nghiệm nghe nhạc mượt mà kể cả khi chạy nền.

English:
SONA is an Android offline music player that allows users to:
 - Play local music files
 - View synchronized lyrics
 - Get personalized recommendations (For You)
 - See real-time trending songs
 - Manage personal playlists
The app is built with modern architecture using Media3 + MediaSessionService for smooth background playback.
